Usage Guidelines

✓ Do

  • Use the dark version as the primary logo
  • Maintain minimum clear space equal to the height of one C bar around the mark
  • Use the transparent mark SVG on custom background colors
  • Scale from SVG for any size — never stretch or distort
  • Use the stacked version for square social formats
  • Use the horizontal version for wide banner formats
  • Use Inter Bold 700 for any additional wordmark uses

✗ Don't

  • Recolor the four C bars — they encode the game mechanic
  • Use the wordmark without the C mark in primary contexts
  • Place on busy photographic backgrounds
  • Add drop shadows, glows, or gradients to the mark
  • Rotate or skew the logo
  • Use the horizontal lockup at smaller than 200px wide
  • Alter the tagline text or spacing
